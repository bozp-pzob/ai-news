# ElizaOS Report – 2025-03-30

## 1. Recent Issues and Fixes in ElizaOS CLI and Twitter Integration

### Twitter API Duplicate Tweet Error
- Issue: Twitter API error 187 due to consecutive duplicate tweets.
- Fix: Prevents identical tweets from being sent back-to-back.
- Tracked in: [GitHub Issue #4086](https://github.com/elizaOS/eliza/issues/4086)
- Resolved in: [Pull Request #4111](https://github.com/elizaOS/eliza/pull/4111)

### NPM Installation Failure
- Problem: Installation of `@elizaos/cli` fails due to missing dependency `@elizaos/plugin-sql@^0.25.6`.
- Cause: Only pre-release versions (1.0.0-beta) are available.
- Suggested Fix: Update `package.json` to reference a valid version.
- Tracked in: [GitHub Issue #4109](https://github.com/elizaOS/eliza/issues/4109)

### Agent Initialization Bug
- Error: `'agents already exists'` when running `npx elizaos create` followed by `npx elizaos start`.
- Environment: Node.js v23.7.0, Debian 12.
- Tracked in: [GitHub Issue #4107](https://github.com/elizaOS/eliza/issues/4107)
